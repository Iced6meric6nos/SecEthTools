    OSI Model

  • 1. Physical Physical data transmission meda Electrical, optical and wireless signals>
  • 2. Data Link Layer Reliable data transfer between adjacent nodes Ethernet (802.3), WiFi (802.11)
  • 3. Network Layer Logical addressing and routing between networks IP, ICMP, IPSec, VPN
  • 4. Transport Layer End-to-end communication and data segmentation UDP TCP
  • 5. Session Layer Establishing, maintaining, and synchronising sessions NFS, RPC
  • 6. Presentation Layer Data encoding, encryption, and compression Unicode, MIME, JPEG, PNG, MPEG
  • 7. Application Layer Providing services and interfaces to applications HTTP, FTP, DNS, POP3, SMTP, IMAP, DHCP
